# 📱 qr_code_generator

**Turn any link into a scannable QR code — in 4 lines of Python.**

A simple yet powerful Python script that takes any URL and instantly generates a clean, downloadable QR code image — no websites, no ads, no uploading your links to some random third-party tool.

---

## ⚡ Why This Exists

Every time you need a QR code, you end up Googling some sketchy "free QR generator" website, dodging ads, and hoping it doesn't track the link you just pasted. This script skips all of that — generate QR codes locally, instantly, and privately, right from your terminal.

---

---

## 🛠️ Usage

```bash
git clone https://github.com/deepanshu-bisht-dev/qr_code_generator.git
cd qr_code_generator
pip install qrcode
python main.py
```

You'll be prompted for:
- The URL you want to convert
- The filename you want to save the QR code as

The image is saved directly in the same folder — ready to share, print, or scan.
